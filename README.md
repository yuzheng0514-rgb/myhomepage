# 董雨铮 · Yuzheng Dong

Bilingual academic homepage for Yuzheng Dong, a Ph.D. student at the Institute of Statistics and Big Data, Renmin University of China. The visual structure is inspired by al-folio and is configured for static deployment to GitHub Pages.

Research interests: optimal transport, data subsampling, and time series.

## GitHub Pages

Push the repository to GitHub and enable **Settings → Pages → Source: GitHub Actions**. The included workflow builds and deploys the static site automatically from the `main` branch.
